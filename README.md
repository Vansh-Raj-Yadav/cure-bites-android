# Cure Bites 🩺🥗

An all-in-one personalized nutrition and telehealth Android application. Cure Bites bridges the gap between medical advice, dietary planning, and food delivery. 

Users can upload their physical descriptions or medical notes from doctors/dietitians, and the app generates a highly personalized meal plan tailored to specific dietary restrictions (e.g., veg/non-veg, allergies). 

## ✨ Key Features
* **Personalized Meal Planning:** Automated diet plans based on user health data and medical notes.
* **Integrated Food Delivery:** Direct connection with vendors to deliver the recommended meals right to the user's door.
* **Telehealth Consultations:** Built-in calling and messaging features to consult directly with doctors and dietitians.
